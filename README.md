# 💰 Tip Calculator

A simple and user-friendly Tip Calculator web application that helps users quickly calculate the tip amount and total bill based on the entered bill amount and tip percentage.

## 🚀 Features

* Enter bill amount
* Select or enter tip percentage
* Automatically calculates tip amount
* Displays total amount to be paid
* Simple and responsive UI

## 🛠️ Built With

* HTML
* CSS
* JavaScript

## 💡 How It Works

1. Enter the total bill amount.
2. Enter or select the tip percentage.
3. Click on the **Calculate** button.
4. The app will display:

   * Tip Amount
   * Total Bill (including tip)

## ▶️ How to Run the Project

1. Download or clone the repository

   ```
   git clone https://github.com/your-username/tip-calculator.git
   ```
2. Open the folder
3. Open `index.html` in your browser
